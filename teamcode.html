TeamCode is a new initiative for all graduates of the Refcode coding class to give them real-world experience collaborating as a team to build a website. Graduates from both the 2017 and 2018 cohorts plan to participate. The first project it to rebuild refcode.org from scratch, so check back soon to see a new and improved website built by Refcode grads!

TeamCode team mates will meet once per sprint (sprints will probably be two weeks) for a sprint planning where we will create tasks for work to be done, estimate them, and assign them according to how much time each team member has. The team will coordinate throughout the sprint using Slack, in-person office hours, and GitHub. Code will be checked into GitHub and reviewed by other team members who can then merge work into the main repository.
